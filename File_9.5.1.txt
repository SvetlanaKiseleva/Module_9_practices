Для создания новой ветки в Git используется команда git branch, после которой следует имя создаваемой ветки.
Команда git branch --list. Она выведет список веток, которые существуют в вашем локальном репозитории. Текущая ветка master будет выделена зелёным цветом.
Команда git branch --list -v. Вы увидите информацию о последних коммитах каждой из веток.
